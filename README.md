# 동대문구 · 서울특별시

> 동대문구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-dongdaemun.github.io/**

동대문구는 서울 동부 도심과 동북권을 잇는 길목에 자리한 자치구로, 구 이름은 한양도성의 동쪽 대문인 흥인지문(속칭 동대문)에서 비롯되었다. 정작 흥인지문 자체는 종로구에 있지만, 옛 도성 동편 바깥에 형성된 생활권을 아우른다는 뜻에서 지명이 굳어졌다. 청량리와 경동시장을 중심으로 한 대규모 재래 상권, 그리고 경희대·한국외대·서울시립대가 몰린 대학가가 이 구를 대표하는 두 얼굴이다. 한약재 유통의 심장인 경동약령시, 서울 최대급 청과·수산 도매 상권이 좁은 면적 안에 촘촘히 들어차 있어 도매와 소매가 하루 종일 뒤섞이는 활기찬 시장 도시의 성격을 띤다. 여기에 세종대왕기념관, 홍릉수목원, 선농단 같은 조선 왕실·근대 유산이 곳곳에 남아 있어 서민 상권과 역사 공간이 이웃한 독특한 풍경을 만든다.
